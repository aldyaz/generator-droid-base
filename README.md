# generator-droid-base

Yeoman generator for [this](https://github.com/aldyaz/skeleton-android-project) Android project

## Usage

- Install yeoman

```bash
npm install -g yo
```

- Install generator

```bash
npm install -g generator-droid-base
```

- Create your Android project directory

```bash
mkdir MyAndroidApp
cd /MyAndroidApp
```

- Then run generator

```bash
yo droid-base
```
